# arcs — 医療AI製品スケジュール

卵子AI／精子AI／ICSI自動化 の WBS スケジュール（ガントチャート）を
ブラウザで管理・閲覧するための静的サイトです。

## ビュー構成

| ページ | 内容 |
|---|---|
| `index.html` | **全体ビュー** — 3製品を1画面で横断確認 |
| `ranshi.html` | 🥚 卵子AI 詳細ビュー |
| `seishi.html` | 🧬 精子AI 詳細ビュー（営業・事業／臨床開発／薬事品証／開発で細分化） |
| `icsi.html` | 🔬 ICSI自動化 詳細ビュー |

各ページ上部のタブで相互に行き来できます。

## 主な機能

- 24ヶ月（2026年度〜2027年度）のガントチャート表示
- カテゴリ（営業／薬事／開発…）ごとの色分け
- ステータス信号（順調／警戒／**遅延**／進行中／未確定）でのフィルタ
- マイルストーン（★）のイベント行表示
- **遅延の可視化**：当初計画（破線ゴースト）と遅延バー（赤枠＋⏰）を併記
- **リスクコメント**：テーマ単位で高/中/低のリスクを記録・表示
- **更新ログ**：変更履歴を画面下部に時系列表示
- TODAYライン、ホバーで詳細ツールチップ
- 印刷対応（A2横）

## 編集方法

スケジュールの更新は **`data/` 配下のデータファイルを編集するだけ** です。
詳しい手順（遅延・リスクの記録方法を含む）は **[UPDATE_GUIDE.md](UPDATE_GUIDE.md)** を参照してください。

```
data/schedule.js        全体ビュー＋イベント＋サマリ＋更新ログ
data/ranshi-detail.js   卵子AI 詳細
data/seishi-detail.js   精子AI 詳細
data/icsi-detail.js     ICSI自動化 詳細
```

## ローカルでの確認

```bash
python3 -m http.server 8000
# http://localhost:8000/ を開く
```

> `file://` で直接開くとデータファイル（`data/*.js`）の読み込みが
> ブラウザにブロックされる場合があります。HTTPサーバ経由で開いてください。
